Github + Hexo 搭建个人博客教程

看到别人有 个人博客属实羡慕，但苦于技术有限，什么前端后端、css，最重要的还得要服务器、域名、备案等等一大堆东西，搞得头都要昏。这些对于小白来说真的太难了/(ㄒoㄒ)/~~，那有没有不需要购买什么服务器、域名、小白就能搭的博客呢？

一、准备工作

1. GitHub pages

  • 注册 Gtihub 帐号
  • 创建仓库

1.1 注册 GitHub 帐号(跳过)

1.2 创建一个仓库

打开 GitHub 并登录，点击头像旁的 ‘+’号

选择 new repository

注意仓库名应这样填写: 用户名.github.io,这就是你的域名，在浏览器地址栏输入，即可访问你的博客。然后点击底部 Create repository 完成创建。

2. 安装运行环境

  • git官网：https://git-scm.com/

  • nodejs官网：https://nodejs.org/en/

2.1 安装 git

打开官网，选择版本。点击下载，下载完毕后，按提示安装即可。（不懂如何安装可以百度）

安装完毕后，打开 cmd 输入 git –version
若出现版本号说明成功

若提示“不是外部命令，也不是可执行程序“,需要手动配置环境变量。

  • 右击 桌面计算机图标 –>> 选择 属性 –>> 找到并点击 高级系统设置

  • 点击 高级 点击下面的 环境变量

  • 选中 ‘PATH’变量 点击 编辑

  • 再点击 新建 将 git 目录下的 bin 文件夹路径输入进去，点击确定即可。

2.2安装 nodejs

去官网下载 nodejs，下载完安装。完毕后打开cmd 输入 node -v & npm -v，若出现版本号则成功.

二、安装Hexo

1.在某个盘新建一个 hexo 文件夹，进入该文件夹中，右击鼠标选择 git bash here

2.在git 窗口中输入 npm inatsll -g hexo-cli 进行安装，完毕后输入 hexo -v 若出现版本号则安装成功。

3.初始化 hexo

输入 hexo init 等待结束

4.启动hexo

输入 hexo s
启动本地服务器，打开浏览器，地址栏输入http://localhost:4000.com 即可访问 hexo

三、部署到GitHub

1.修改配置文件_config.yml

打开 github 点击你的头像，选择 Your repositionies,进入你新建的仓库，复制你的仓库地址

进入 hexo 文件夹，找到并打开 _config.yml**文件，直接拉到最下面修改 ‘# Deployment 部分’

这是修改后的
deploy:
type: git
repo: 这里填写你复制的仓库地址
branch: master
