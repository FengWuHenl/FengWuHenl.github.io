Github + Hexo 搭建个人博客

看到别人搭建的个人博客属实羡慕，但苦于技术有限，什么前端后端、css，最重要的还得要服务器、域名、备案等等一大堆东西，搞得头都要昏。这些对于小白来说真的太难了/(ㄒoㄒ)/~~，那有没有什么不需要购买服务器、域名、就能搭的博客呢？当然有！~它就是 Hexo ！

什么是 Hexo？

  • hexo 是一款基于 Node.js 简洁高效的静态博客框架，可以方便的生成静态网页托管在 GitHub 上，大家可以进入 hexo官网 进行详细的查看.

一、准备工作

1. GitHub pages

  • 注册 Gtihub 帐号
  • 创建仓库

1.1 注册 GitHub 帐号(跳过)

1.2 创建一个仓库

打开 GitHub 并登录，点击头像旁的 ‘+’号

选择 new repository

注意仓库名应这样填写: 用户名.github.io,这就是你的域名，在浏览器地址栏输入，即可访问你的博客。然后点击底部 Create repository 完成创建。

注意这里把你的仓库地址复制下来留着备用

2. 安装运行环境

  • git官网：https://git-scm.com/

  • nodejs官网：https://nodejs.org/en/

2.1 安装 git

打开官网，选择版本。点击下载，下载完毕后，按提示安装即可。（不懂如何安装可以百度）

安装完毕后，打开 cmd 输入 git —-version (注意这里 version 前面是两条相连的杠，中间没有空格。以图为准)
若出现版本号说明成功

若提示“不是外部命令，也不是可执行程序“,需要手动配置环境变量。

  • 右击 桌面计算机图标 –>> 选择 属性 –>> 找到并点击 高级系统设置

  • 点击 高级 点击下面的 环境变量

  • 选中 PATH变量 点击 编辑

  • 再点击 新建 将 git 目录下的 bin 文件夹路径输入进去，点击确定即可。

    再试一下，若依然没有版本号重启一下电脑试试。

2.2安装 nodejs

去官网下载 nodejs，下载完安装。完毕后打开cmd 输入 node -v & npm -v，若出现版本号则成功.

二、安装Hexo

1.在某个盘新建一个 hexo 文件夹，用于存放 hexo文件 。进入该文件夹中，右击鼠标选择 git bash here

2.在git 窗口中输入 npm inatsll -g hexo-cli 进行安装，完毕后输入 hexo -v 若出现版本号则安装成功。

3.初始化 hexo

输入 hexo init 等待结束

4.启动hexo

输入 hexo s
启动本地服务器，打开浏览器，地址栏输入http://localhost:4000.com 即可访问 hexo

三、部署到 GitHub

1. 设置 ssh

输入命令 cd ~/.ssh

再次输入 ssh-keygen -t rsa -C ‘你注册github用的邮箱’

接下来我们去C盘【路径 C:\用户\你的用户名\ .ssh】

使用记事本打开 id_rsa.pub 文件，并复制生成的 ssh key

然后打开浏览器在已登录的 Github主页 ，点击右侧 头像 选择 settings

点击 SSH and GPG keys

再点击 New SSH key

最后再测试一下，是否添加成功？输入命令 ssh -T git@github.com 期间会问你是否继续？输入 yes 即可。看到 “Hi，用户名…successfully..“ 说明成功！

接着再设置你的 GitHub 帐号信息

git config –global user.name “github用户名”
git config –global user.email “注册github使用的邮箱”

2. 安装上传工具

输入命令 npm install hexo-deployer-git —-save(注意这里 save 前面是两条相连的杠 - -，中间没有空格，我为了凸显加了空格。不加看起来就跟一条杠一样。以图为准)

3.修改 hexo 配置文件_config.yml

进入 hexo 文件夹内，找到并打开 _config.yml 文件，直接拉到最下面修改 ‘ # Deployment 部分’ 这里把你之前复制的仓库地址添加到配置文件中:

deploy:
  type: git
  repo: 这里填写你复制的仓库地址
branch: master

4. 部署到 GitHub 上

输入命令 hexo d -g 部署到 Github 上 (注意：要在 hexo 目录下执行)

此时再回到浏览器刷新一下，你会发现仓库里多了一些文件

在浏览器地址栏输入你的 用户名.github.io 就可以访问你的博客啦~ 如何发文章换主题详见 官方文档
